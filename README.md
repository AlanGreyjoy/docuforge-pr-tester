# DocuForge PR Tester

A simple Node.js server using Express.

## Setup

1. Install dependencies:

```
npm install
```

2. Create a `.env` file in the root directory with the following variables:

```
PORT=3000
```

## Running the Server

### Development mode

```
npm run dev
```

### Production mode

```
npm start
```

## API Endpoints

- `GET /`: Returns a welcome message
